# UNO Online

基于 Node.js 和 WebSocket 构建的实时多人在线 UNO 网页游戏。采用原生的毛玻璃（Glassmorphism）暗色 UI 设计，支持实时跨设备游玩、智能人机对战、观战模式以及独创的 QTE 结算机制。零前端框架依赖，即开即玩。

## ✨ 核心特性

### 🎮 丰富的房间与游戏系统
* **灵活的房间设置**：支持创建 2-12 人的游戏房间，可自由配置真实玩家与 AI 机器人的比例。
* **多途径加入**：支持通过 3 位数房间号加入、大厅公开房间列表一键加入，或通过专属链接（`/:roomId`）直接分享邀请。
* **观战模式 (Spectator Mode)**：当房间满员或对局已开始时加入，将自动切换为观战者，可自由切换视角旁观不同玩家的手牌，并参与房间聊天。
* **断线与防挂机机制**：5 分钟（AFK）无操作自动踢出；游戏进行中玩家若离线，将无缝交由 AI 机器人接管当前对局。

### 🤖 智能 AI 机器人
* **内置人机**：服务器内置了丰富的随机 AI 昵称（如“半盏流年”、“259”等）。
* **拟真逻辑**：机器人拥有独立的发牌和出牌逻辑，并且在触发 QTE 时具备随机的拟人化延迟反应时间，确保游戏体验的真实感。

### ⚡ 特色交互与玩法
* **独创 QTE 抢答机制 (Quick Time Event)**：当某位玩家打出最后一张牌（手牌清空）时，并非立即获胜，而是触发全场 QTE。系统会下发随机校验码（如 `UNO-123`），手牌清空者输入成功方可夺冠；若被其他玩家率先输入抢答抓获，则被罚抽 2 张牌并继续游戏。
* **全方位互动**：
    * **文字聊天**：支持房间内实时聊天，观战者发言会带有专属 `[观战]` 标签。
    * **“催促”功能**：内置带冷却时间的催促按钮，触发时带有专属音效提示与设备屏幕震动特效。
* **隐藏彩蛋 (Cheat Mode)**：连续点击页面底部版权信息 10 次可激活“作弊模式”。在该模式下，玩家可以无视规则出牌，并将不合法卡牌强行变身为主导战局的超级 +4 黑牌。

### 🎨 现代化的 UI / UX
* **毛玻璃暗黑美学**：全局采用流行的暗色系搭配半透明背景模糊（Backdrop-filter）效果。
* **流畅的动效设计**：从卡牌抽取、出牌飞行轨迹、回合高亮切换到胜利皇冠动画，均配备了细腻的 CSS 动画过渡。
* **全端适配**：通过媒体查询深度优化了移动端体验（手牌缩放、重排版 HUD 等），支持手机浏览器无缝对战。

## 🛠️ 技术栈

* **后端**：
    * [Node.js](https://nodejs.org/) 
    * [Express](https://expressjs.com/) (静态资源服务与路由)
    * [ws](https://github.com/websockets/ws) (WebSocket 实时双向通信)
* **前端**：
    * HTML5 / CSS3 / Vanilla JavaScript
    * 使用 CSS Variables 实现主题控制，原生 SVG 绘制卡牌图案。
